2 App Description

Copy my Data is an iOS application developed by Gaming Geek Studio that enables users to transfer personal data — including contacts, photos, videos, and calendar events — between devices. Transfers occur via:

  • WiFi Transfer — peer-to-peer over a local network using Bonjour/mDNS (no internet required)
  • Link Transfer — via a temporary cloud relay for devices on different networks

The App is available for iPhone and iPad running iOS 15.6 or later. Features may vary between devices and iOS versions.
